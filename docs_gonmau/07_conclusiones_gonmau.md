# Conclusiones técnicas y jurídicas

El ataque contra Sony Pictures Entertainment evidenció cómo un incidente informático puede transformarse simultáneamente en:

- una crisis operacional
- un problema jurídico
- un conflicto reputacional
- una amenaza internacional

Desde la perspectiva técnica, el caso demostró el impacto de malware destructivo capaz de:

- comprometer confidencialidad
- destruir integridad de datos
- afectar disponibilidad de sistemas

## Conclusiones jurídicas

La Ley 21.459 resulta plenamente aplicable respecto de:

- acceso ilícito
- ataque a integridad de datos
- afectación de sistemas informáticos

Asimismo, la Ley 19.628 adquiere relevancia debido a la exposición de información personal de trabajadores y documentos internos.

El caso también demuestra la importancia de cooperación internacional frente a delitos informáticos transnacionales.

## Recomendaciones de seguridad

**Fortalecimiento técnico**
- segmentación de redes
- monitoreo continuo
- autenticación multifactor
- gestión de privilegios
- respaldo seguro y desconectado
- detección temprana de amenazas

**Gestión organizacional**
- planes de respuesta a incidentes
- capacitación continua
- gobierno corporativo de ciberseguridad
- auditorías periódicas

**Protección de datos**
- clasificación de información
- minimización de datos
- cifrado
- controles de acceso estrictos

## Reflexión crítica

El caso Sony Pictures demuestra que la ciberseguridad no puede considerarse únicamente un problema tecnológico.

La protección de la información requiere integración entre:

- seguridad técnica
- cumplimiento normativo
- gestión organizacional
- cultura de seguridad

Además, el incidente evidencia que las consecuencias de una brecha no se limitan al robo de información, sino que pueden afectar:

- derechos fundamentales
- estabilidad empresarial
- confianza pública
- relaciones internacionales

Por ello, la prevención, preparación y resiliencia organizacional constituyen elementos esenciales dentro de cualquier estrategia moderna de seguridad de la información.
