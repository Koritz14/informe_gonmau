# Responsabilidades del ataque
## 1. Responsabilidades derivadas del ataque

**Responsabilidad penal**

Los responsables podrían enfrentar imputaciones por:

- acceso ilícito
- destrucción de información
- sabotaje informático
- divulgación ilícita de datos

**Normas relacionadas**

- Ley 21.459
- Convenio de Budapest

## 2. Responsabilidad de la empresa
Sony Pictures Entertainment

**Responsabilidad civil**

La empresa podría enfrentar acciones derivadas de:

- exposición de datos personales
- afectación a trabajadores
- posibles daños económicos

**Responsabilidad administrativa**

Desde una perspectiva moderna de cumplimiento, las organizaciones tienen deberes de:

- protección de información
- gestión de riesgos
- implementación de controles de seguridad

## 3. Responsabilidad de ejecutivos

Los ejecutivos podrían enfrentar cuestionamientos respecto a:

- gobernanza de ciberseguridad
- gestión de riesgos
- protección de información corporativa

No obstante, las fuentes no atribuyen responsabilidad penal individual específica a directivos.

## 4. Responsabilidad de terceros

Podrían existir responsabilidades asociadas a:

- proveedores tecnológicos,
- contratistas,
- empresas de soporte.

Sin embargo, las fuentes analizadas no identifican participación directa comprobada.

| Actor      | Penal                              | Civil   | Administrativa |
| -- | -- | -- | -- |
| Atacantes  | Sí                                 | Sí      | Posible        |
| Empresa    | No directa como autora del ataque  | Sí      | Sí             |
| Ejecutivos | No acreditada                      | Posible | Posible        |
| Terceros   | No acreditada                      | Posible | Posible        |

**Consideración jurídica relevante**

En incidentes modernos de ciberseguridad, la responsabilidad no se limita únicamente al atacante.

También se evalúa:

- diligencia organizacional
- controles implementados
- capacidad de respuesta
- protección de datos personales
